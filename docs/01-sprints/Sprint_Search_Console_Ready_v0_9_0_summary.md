# Sprint — SEO Semântico / Search Console Ready v0.9.0

Status: concluída e consolidada em histórico.

## Objetivo

Criar uma camada operacional para facilitar a submissão, acompanhamento e priorização de URLs no Google Search Console sem uso indevido da Google Indexing API.

## Escopo

- Processamento SEO operacional.
- Janela de pendências recentes.
- Listas PT-BR e EN-US prontas para Search Console.
- Ações rápidas: abrir post, copiar URL e abrir Search Console.
- Marcação de prioridade editorial.
- Fluxo seguro para indexação manual.

## Módulos afetados

- M360 Editorial.
- M360 Admin.
- M360 Semantic Relations.

## Decisões

- Não usar Google Indexing API para conteúdo editorial comum.
- Priorizar fluxo assistido para Search Console.
- Separar URLs PT-BR e EN-US.
- Usar a camada semântica para priorizar páginas com maior valor editorial.

## Critérios de aceite

- URLs recentes listadas por idioma.
- Botão de cópia funcional.
- Botão de abertura no Search Console funcional.
- Critérios de prioridade editorial exibidos.
- Fluxo compatível com boas práticas de indexação.

## Resultado

A sprint consolidou uma operação mais segura e rastreável para indexação e acompanhamento editorial no Search Console, alinhada à estratégia de SEO semântico do projeto.

## Observação de migração

O arquivo original disponível no pacote local estava em formato binário do Word/Docs apesar do nome sem extensão. Esta versão registra o resumo técnico canônico para manutenção futura.
