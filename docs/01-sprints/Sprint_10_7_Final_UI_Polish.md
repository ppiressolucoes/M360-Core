# Sprint 10.7 - Final UI Polish

Status: pronto para execucao do workflow
Versao: 0.4.1.2
Fluxo: GitHub First

## Objetivo

Fechar a sprint de refinamento visual com ajustes finos no menu, breadcrumb, espaçamento e badges das views dinamicas do M360 Core.

## Escopo

- Novo modulo CSS `m360-ui-polish.css`.
- Indicador de HOME mais limpo.
- Indicador de submenu mais discreto.
- Separadores do menu com menor contraste.
- Breadcrumb com destaque atual mais sutil.
- Hover refinado no breadcrumb.
- Espacamento vertical ajustado entre menu, breadcrumb e conteudo.
- Contadores de Author, Category e Tag refinados.
- Suavizacao de icones de metadados em posts.
- Ajuste mobile de padding nas views dinamicas.

## Arquivos adicionados

- plugin/assets/css/m360-ui-polish.css

## Arquivos atualizados

- plugin/includes/class-m360-core.php
- plugin/m360-core.php
- VERSION.md

## Compatibilidade considerada

- Tema WordPress do portal de noticias.
- Mengao 360 Bolao.
- M360 Home Editorial.
- M360 Semantic Relations.
- Elementor.
- Polylang.
- LiteSpeed Cache.

## Criterios de aceite

- Menu principal com visual mais alinhado ao tema.
- Breadcrumb com item atual destacado sem parecer botao pesado.
- Search, Author, Category e Tag sem regressao.
- EN-US e PT-BR preservados.
- Sem impacto funcional nos plugins Bolao, Home Editorial e Semantic Relations.

## Encerramento

Esta release fecha a Sprint 10.7 de refinamento visual antes das proximas sprints evolutivas.
