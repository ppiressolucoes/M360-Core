# M360 Core v0.5.3.3 — Minimal Search Hero

## Objetivo

Reduzir a altura e a densidade visual do componente de busca no cabeçalho.

## Alterações

- O Search Hero exibe somente o campo de busca por padrão.
- A orientação permanece no placeholder multilíngue do campo.
- O botão continua incorporado ao campo.
- Foram reduzidos padding, raio e sombra do box.
- Título e subtítulo podem ser reativados com `show_intro="true"`.

## Homologação sugerida

- Validar desktop e mobile em PT-BR e EN-US.
- Confirmar que o placeholder não fica encoberto pelo botão.
- Confirmar busca por Enter, pelo botão e bloqueio de valor vazio.
