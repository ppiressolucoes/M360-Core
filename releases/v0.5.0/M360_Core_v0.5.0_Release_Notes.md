# M360 Core v0.5.0 — Ads Manager Slot Management UX

Primeira entrega da linha **Plataforma Comercial M360**.

## Entregas

- cartões de slots agrupados por contexto editorial;
- pesquisa por nome ou identificador;
- filtros por contexto, runtime e ocupação;
- estados visuais Livre e Ocupado;
- contador de resultados e alterações pendentes;
- botão único **Salvar todos os vínculos**;
- gravação transacional validada no banco;
- layout administrativo responsivo.

## Compatibilidade

O renderer público, os shortcodes e o Runtime Mapping homologado na v0.4.4.7 permanecem inalterados.

## Homologação

1. Filtrar slots em cada critério.
2. Alterar múltiplos vínculos em contextos distintos.
3. Confirmar o contador de alterações.
4. Salvar uma única vez e conferir a persistência.
5. Validar a interface em desktop e mobile.
